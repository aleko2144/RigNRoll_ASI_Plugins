# RigNRoll_ASI_Plugins
Rig'n'Roll ASI plugins.

List of plugins:
* RnR_NewGameMod (only R'n'R Gold Edition) - allows to change default game start position and vehicle when new game is started without editing game.cnf;
* RnR_EncryptionHack (only EN Steam version of R'n'R) - allows to load unencrypted pkqt, xml, sch, and cnf files into the game;

Sources:
* "CPatch.h" file is from [D2DInputWrapper](https://github.com/Voron295/rignroll-dinput-wrapper) by [Voron295](https://github.com/Voron295)
